**Interface:**

**State variables:**
- `garmentTypes: GarmentTypeRow[]` — full list
- `loading: boolean` — initial load
- `error: string | null` — load error
- `editingId: string | null` — which row is in edit mode
- `editName: string` — edit form name value
- `editEmoji: string` — edit form emoji value
- `isSubmitting: boolean` — save in progress
- `deleteConfirmId: string | null` — which row has delete confirmation open
- `deleteUsageCount: number | null` — usage count for delete confirmation
- `successMessage: string | null` — auto-dismissing success banner

**Data loading — `loadGarmentTypes()`:**
- `fetch('/api/garment-types', { cache: 'no-store' })`
- Extract `data.garmentTypes` array
- Sort by `display_order` ascending (primary), then `name` ascending (fallback)
- Store in state

**Sort display:**
- Sort `garmentTypes` by `display_order` ASC in render (use a `sortedTypes` variable):
  ```typescript
  const sortedTypes = [...garmentTypes].sort((a, b) => a.display_order - b.display_order);
  ```

**Inline edit — `handleStartEdit(type)`:**
- Set `editingId = type.id`
- Set `editName = type.name`
- Set `editEmoji = type.icon || '📝'`

**Save edit — `handleSaveEdit(typeId)`:**
- `PUT /api/admin/garment-types` with body `{ id: typeId, name: editName.trim(), icon: editEmoji }`
- On success: optimistic update — `setGarmentTypes(prev => prev.map(t => t.id === typeId ? { ...t, name: editName.trim(), icon: editEmoji } : t))`
- Clear edit state, show success message "Type modifie"
- On error: `alert(data.error || 'Erreur lors de la modification')`

**Delete flow — `handleCheckDelete(typeId)`:**
- `GET /api/admin/garment-types?usage=true&id=${typeId}`
- Set `deleteUsageCount = data.usageCount`
- Set `deleteConfirmId = typeId`

**Confirm delete — `handleDelete()`:**
- `DELETE /api/admin/garment-types?id=${deleteConfirmId}`
- On success: remove from local state — `setGarmentTypes(prev => prev.filter(t => t.id !== deleteConfirmId))`
- Show success message "Type supprime"
- Clear delete state
- On error: `alert(data.error || 'Erreur lors de la suppression')`

**Reorder up — `handleMoveUp(type)`:**
- Find index in `sortedTypes`
- If index <= 0, return (already first)
- Get `prevType = sortedTypes[index - 1]`
- Fire two parallel PUT requests swapping `display_order` values:
  ```typescript
  await Promise.all([
    fetch('/api/admin/garment-types', {
      method: 'PUT',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ id: type.id, display_order: prevType.display_order }),
    }),
    fetch('/api/admin/garment-types', {
      method: 'PUT',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ id: prevType.id, display_order: type.display_order }),
    }),
  ]);
  ```
- Optimistic local state update: swap `display_order` values in state

**Reorder down — `handleMoveDown(type)`:**
- Same pattern as moveUp but with `nextType = sortedTypes[index + 1]`

**Success message — `showSuccess(message)`:**
- Set `successMessage = message`
- `setTimeout(() => setSuccessMessage(null), 3000)`
