  <name>Task 1: Install emoji-mart dependencies and create EmojiPicker component</name>
  <files>src/components/ui/emoji-picker.tsx</files>
  <action>
1. Install dependencies:
   ```bash
   npm install @emoji-mart/react @emoji-mart/data
   ```
   Note: `emoji-mart` is a peer dep and will be installed automatically.

2. Create `src/components/ui/emoji-picker.tsx` as a `'use client'` component:

   - Import `dynamic` from `next/dynamic`
   - Import `data` from `@emoji-mart/data` at module scope
   - Import `Popover`, `PopoverTrigger`, `PopoverContent` from `@/components/ui/popover`
   - Import `useState` from `react`

   - Use `const Picker = dynamic(() => import('@emoji-mart/react'), { ssr: false })` — CRITICAL: without `ssr: false`, Next.js will crash with "window is not defined" because emoji-mart registers custom elements using browser APIs.

   - Define interface:
     ```typescript
     interface EmojiPickerProps {
       value: string;           // current emoji displayed on button
       onSelect: (emoji: string) => void;  // called with emoji.native string
       disabled?: boolean;
     }
     ```

   - Export named function `EmojiPicker({ value, onSelect, disabled })`:
     - Internal `open` state via `useState(false)`
     - Return a `Popover` with `open` and `onOpenChange` controlling visibility
     - `PopoverTrigger` renders a `button` (type="button") with:
       - `disabled` prop passed through
       - Classes: `relative w-9 h-9 text-lg flex items-center justify-center rounded border border-border hover:border-primary-400 touch-manipulation min-h-[44px] min-w-[44px]`
       - Shows `{value}` as content
       - Small pencil overlay: `<span className="absolute bottom-0 right-0 text-[9px] bg-white rounded-full leading-none">✏️</span>`
       - `aria-label="Choisir un emoji"`
     - `PopoverContent` with:
       - `className="w-auto p-0 border-0 shadow-xl"` — IMPORTANT: `w-auto` lets the picker control its own width (~352px). Do NOT set a fixed width.
       - `align="start"`
       - Contains `<Picker>` with props:
         - `data={data}`
         - `onEmojiSelect={(emoji: any) => { if (emoji?.native) { onSelect(emoji.native); setOpen(false); } }}` — guard for undefined `native` on non-standard emoji
         - `theme="light"`
         - `locale="fr"` — French search labels
         - `previewPosition="none"` — saves vertical space
         - `skinTonePosition="none"` — simplifies UI

   Do NOT import `@emoji-mart/react` at module scope (only via `dynamic()`). Do NOT store the full emoji object — only `emoji.native` (the Unicode string like "✂️").
