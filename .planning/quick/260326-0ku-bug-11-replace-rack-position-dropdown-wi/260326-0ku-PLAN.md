**1. Create API: `src/app/api/rack-occupancy/route.ts`**

GET endpoint that returns all orders with `rack_position` set and `status` in ('ready', 'delivered'):

```typescript
import { NextResponse } from 'next/server';
import { createServiceRoleClient } from '@/lib/supabase/server';

export const dynamic = 'force-dynamic';

export async function GET() {
  const supabase = await createServiceRoleClient();
  const { data, error } = await supabase
    .from('order')
    .select('id, order_number, rack_position')
    .not('rack_position', 'is', null)
    .in('status', ['ready', 'delivered'])
    .eq('is_archived', false);

  if (error) {
    return NextResponse.json({ error: error.message }, { status: 500 });
  }

  // Return as a map: { "A1": { orderId: "xxx", orderNumber: 123 }, "B3": ... }
  const occupancy: Record<string, { orderId: string; orderNumber: number }> = {};
  for (const row of data || []) {
    if (row.rack_position) {
      occupancy[row.rack_position] = {
        orderId: row.id,
        orderNumber: row.order_number,
      };
    }
  }

  return NextResponse.json({ occupancy });
}
```

- Render a CSS grid with `gridTemplateColumns: 'auto repeat(cols.length, 40px)'` and `gap: '4px'`.
- Top row: empty corner cell + column headers (1-10), each 40px wide, centered, text-xs, text-muted-foreground.
- Each data row: row header (A/B/C) on left (auto width, text-xs, font-medium, centered vertically) + cells.
- Cell states (each cell is a `<button>` element, 40px x 40px, rounded-md, text-xs, font-medium):
  - **Available (empty):** `bg-gray-50 hover:bg-blue-50 text-gray-600 border border-gray-200` -- shows position label (e.g., "A3").
  - **Occupied (other order):** `bg-gray-200 text-gray-500 cursor-pointer` -- shows order number (e.g., "#42"). Still clickable for reassignment.
  - **Selected (current order):** `bg-blue-600 text-white` -- shows position label with accent. Clickable to unassign.
  - **Disabled (not editable):** all cells get `opacity-50 pointer-events-none` when `disabled=true`.
- Track `saving` state (boolean) to disable grid during save.
