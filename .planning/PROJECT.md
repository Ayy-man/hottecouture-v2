# Project: Hotte Couture

**Type:** Tailoring shop management app (Next.js + Supabase)
**Client:** Audrey (Quebec-based seamstress/shop owner)
**Language:** French-first customer-facing, English internal/dev

## Tech Stack
- **Frontend:** Next.js 14 (App Router), React, Tailwind CSS, shadcn/ui
- **Backend:** Next.js API routes, Supabase (Postgres + Auth)
- **Integrations:** Stripe (payments), Google Calendar, Make.com (webhooks to GHL/QuickBooks)
- **Deployment:** Vercel
- **i18n:** next-intl with `locales/fr.json` and `locales/en.json`
